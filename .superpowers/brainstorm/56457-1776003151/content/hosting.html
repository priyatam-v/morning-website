PostHog hosting — Cloud or Self-hosted?

This affects setup complexity and where your data lives.

A

PostHog Cloud (app.posthog.com)

Managed SaaS. Free up to 1M events/month. Fastest setup — just an API key. Data stored on PostHog's servers (US or EU region).

B

Self-hosted

You run PostHog on your own infra (Docker / Kubernetes). Full data ownership, but requires infra maintenance. More setup work.
